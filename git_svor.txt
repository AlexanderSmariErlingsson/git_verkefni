"#Git spurningar og svör"
1. Hvað gera eftirfarandi Linux skipanir?
	cd   Skiptir um directory
	cd ~  Fer á heimasvæði notanda
	ls  Sýnir allar skrár og möppur í núverandi directory
	pwd  Prentar hvaða directory notandi er í
	mkdir  Býr til directory


2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone  Klónar repository í nýtt directory
	git log   Sýnir git history
	git status   Skoðar status af git repository
	git diff  Sýnir mun milli commits eða commit og annara hluta
	git checkout  Switchar branches eða endurnýjar working tree skrár

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff  Sýnir það sem er búið að breytast en ekki komið í indexinn enn
	b) git diff --staged  Sýnir það sem er búið að bæta við indexinn en ekki commitað
	c) git diff commit1 commit2	Sýnir breytingar milli commit1 og commit2

4. Hvað er útgáfustýring (e. version control)?   Umsjón breytinga við skjöl, tölvuforrit, stórar vefsíður og önnur gagnasöfn.

5. Hverjir eru helstu kostir við að nota GIT?   Hópvinnumöguleikar.

6. Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?   Brjálað oft en ekki missa þig, ekki gera það óþarfa erfitt að leita að fyrrverandi verkefni en ekki sóa tíma við að endurgera allt.
